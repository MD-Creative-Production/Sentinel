# Monitoring Module

Comprehensive security monitoring capabilities including:

- **Multisig Monitoring**: Track multisig wallet activities, signers, approvals, and threshold changes
- Real-time change detection
- Security alert generation

## Modules

### Multisig Monitoring

See [multisig/README.md](multisig/README.md) for detailed documentation.

## Features

- Signer tracking and audit trail
- Approval workflow monitoring
- Threshold change detection with alerts
- Event logging for compliance
- Real-time notifications on critical changes
- Integration with notification providers (Discord, Telegram)

## Usage

All monitoring modules are automatically imported in the AppModule and provide REST APIs for integration with other components.
